Halmos

  • Naive Set Theory, by Paul Halmos
  • Quite a short book, which is unsual for a math textbook
  • This makes it good for a reference to set theory, and as a light exposure to it
  • Unfortunately, there are almost no exercises

Pinter

  • A Book of Set Theory, by Charles C. Pinter
  • Quite a dense book, and has the option to be less formal than other books
  • Extensive catalog of exercises of all difficulty
  • Quite good for beginners

Jech

  • Set Theory, by Thomas Jech
  • Very advanced book, contains lots more material than most other set theory books. (Chapter 1 of 38 covers most that other books do)
  • Not recommended for beginners, but for more advanced exploration into Set Theory this book is perfect
